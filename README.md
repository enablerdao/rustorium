## 🚀 クイックスタート

### 📦 インストール
```bash
curl -L https://get.rustorium.org | bash
```

### 🎮 開発環境の起動
```bash
# マルチノードテストネット
rustorium dev --nodes 5

# シングルノード（本番モード）
rustorium start --release

# 開発サーバー
rustorium serve --dev
```

### 🔧 設定例
```bash
# バリデーターとして起動
rustorium validator --stake 100000

# シャードノードの追加
rustorium shard create --name "my-shard"

# メトリクスの確認
rustorium metrics --watch
```
